**Описание**

Сайт предоставляет возможность делится фотографиями. 
Для примера представлено несколько фотографий и то как они выглядят в учетной записи пользователя. 
На данный момент в проэкте не хватаят JavaScript для различных действи при нажатии на кнопки
В проекте используются Методология БЭМ, Flex, Grid, он адаптивен благодаря @media.
